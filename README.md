### SmartAdmin Login Assistant Extension
A Chrome extension to help UParcel admins log in to the SmartAdmin PC app directly from their browser. This extension supports the SmartAdmin AI-powered admin dashboard capstone project.
- Seamless browser-to-app login
- Auto-save and auto-fill credentials
- TOTP code generation
- Privacy-first: all data stored locally
