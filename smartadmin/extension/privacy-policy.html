Privacy Policy

Last Updated: November 2025

SmartAdmin Login Assistant ("the Extension") is committed to protecting your privacy. This privacy policy explains how we handle your data.

1. Data Collection and Usage

The Extension collects and stores the following data locally in your browser:

  • Username and Password: Automatically saved when you log in to SmartAdmin sites
  • TOTP Secret: Saved from your profile page for two-factor authentication code generation
  • Extension Settings: Your preferences including:
    • Auto-login toggle
    • Auto-fill OTP toggle
    • Remember credentials setting
    • Clear credentials after login option
  • Login Status: Tracks whether you're currently logged in and which environment (staging/production)

2. Data Storage

All data is stored exclusively in your browser using Chrome's secure storage API (chrome.storage.local).

Important Points:

  • ❌ Data is NOT transmitted to any external servers
  • ❌ Data is NOT shared with any third parties
  • ❌ Data is NOT sold, rented, or monetized in any way
  • ❌ We do NOT use analytics, tracking, or advertising services
  • ✅ All data remains on your device only
  • ✅ You have full control and can clear data anytime

3. How Data is Used

The Extension uses your stored data exclusively for:

  • Auto-filling login forms on SmartAdmin pages when enabled
  • Generating TOTP codes for two-factor authentication
  • Deep-link authentication to launch the SmartAdmin desktop application with credentials
  • Applying your preferences (auto-login, cleanup policies, etc.)

4. Website Access

The Extension only accesses the following domains:

  • stg.uparcel.sg - Staging environment
  • www.uparcel.sg - Production environment

Specifically, the Extension interacts with:

  • Login pages (/admin/login) - To save and fill credentials
  • OTP pages (/admin/otp/) - To generate and fill TOTP codes
  • Profile pages (/admin/profile/) - To save TOTP secrets
  • Admin home pages (/admin/) - For post-login actions

5. Third-Party Services

This Extension does NOT use:

  • Analytics services (e.g., Google Analytics)
  • Tracking or advertising networks
  • External APIs or cloud services
  • Social media integrations

The Extension operates entirely offline and client-side.

6. Data Deletion

You can delete all stored data at any time using any of these methods:

Method 1: Extension Settings

  1. Click the Extension icon in your browser toolbar
  2. Click the gear icon (⚙️) to open Settings
  3. Click "Clear" next to "Clear Credentials"

Method 2: Uninstall Extension

Uninstalling the Extension automatically removes all stored data from your browser.

Method 3: Browser Settings

You can also clear extension data through Chrome's site settings for the Extension.

7. Permissions Explanation

The Extension requests the following permissions:

Storage Permission

Purpose: Required to save your credentials, TOTP secret, and settings locally in your browser's secure storage.

Host Permissions (stg.uparcel.sg, www.uparcel.sg)

Purpose: Required to:

  • Detect when you're on SmartAdmin login pages
  • Auto-save credentials when you log in
  • Auto-fill username, password, and TOTP codes
  • Enable deep-link login to the desktop application

8. Security

We take security seriously:

  • All data is stored using Chrome's secure storage API
  • Credentials are stored locally and never transmitted
  • The Extension uses HTTPS-only connections
  • Source code is available on GitHub for transparency
  • Regular updates to address security issues

9. Children's Privacy

This Extension is not directed at children under 13. We do not knowingly collect data from children.

10. Changes to This Privacy Policy

We may update this privacy policy from time to time. When we do:

  • The "Last Updated" date at the top will be changed
  • Material changes will be highlighted in the Extension's update notes
  • Continued use of the Extension after changes constitutes acceptance

11. Open Source

This Extension is open source under the MIT License. You can review the complete source code on GitHub to verify our privacy practices.

12. Your Rights

You have the right to:

  • Access all data stored by the Extension (view in Extension popup)
  • Delete your data at any time (via Settings or uninstall)
  • Disable specific features (auto-login, auto-fill, etc.)
  • Stop using the Extension at any time
